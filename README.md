# MansBook

## Тема проекта
Онлайн-библиотека / книжный клуб

Описание
MansBook - сайт книжного клуба для любителей литературы. На сайте представлен каталог книг для чтения и обсуждения, информация о клубе и контактные данные.

Страницы сайта
- Главная (index.html)
- О клубе (pages/about.html)
- Каталог (pages/catalog.html)
- Контакты (pages/contacts.html)
